About Drawmatic

Drawmatic is my final project for Harvard's CS50 course. You too can enjoy this course by heading over to the CS50 website.

Check out the github repository for this project to find out how I built it and what my plans for future releases are. You should also clone it and build your own thing!

Drawmatic uses Google's tensorflowjs hand pose model and is built using Bootstrap.
